src.myla_game.skill module

ゲーム内に登場するすべてのスキル（武器スキル、ジョブスキルなど）を定義するモジュール。

Skill Enumは、各スキルのID、名前、トリガー条件、そして具体的な効果を実装した関数を 一元管理します。これにより、スキルデータとロジックが密結合し、管理が容易になります。

class src.myla_game.skill.Skill(value)[ソース]

ベースクラス: Enum

ゲーム内のすべてのスキルを定義するEnum。

各メンバーは以下の情報を持つタプルとして定義されます: - id (int): スキルの一意なID。 - name_ja (str): スキル名（日本語）。 - trigger_type (list[TriggerType]): スキルが発動する条件のリスト。 - func (function): スキルの効果を実装した関数。

各スキルロジック関数は、`action_data.ActionData`オブジェクトを引数に取り、 戦闘状態の変更や追加効果の発生などを行います。

ASSASSIN_1 = (10, 'アサシン１', [TriggerType.DEFEAT], <function Skill.func_assassin_1>)
ASSASSIN_2 = (11, 'アサシン２', [TriggerType.CRIT], <function Skill.func_assassin_2>)
BERSERKER_1 = (8, 'バーサーカー１', [TriggerType.CRIT], <function Skill.func_berserker_1>)
BERSERKER_2 = (9, 'バーサーカー２', [TriggerType.DAMAGED, TriggerType.HEALED], <function Skill.func_berserker_2>)
CLERIC_1 = (6, 'クレリック１', [TriggerType.CRIT], <function Skill.func_cleric_1>)
CLERIC_2 = (7, 'クレリック２', [TriggerType.CRIT], <function Skill.func_cleric_2>)
FIGHTER_1 = (2, 'ファイター１', [TriggerType.CRIT], <function Skill.func_fighter_1>)
FIGHTER_2 = (3, 'ファイター２', [TriggerType.ATTACK], <function Skill.func_fighter_2>)
GUARDIAN_1 = (4, 'ガーディアン１', [TriggerType.ATTACK], <function Skill.func_guardian_1>)
GUARDIAN_2 = (5, 'ガーディアン２', [TriggerType.CRIT], <function Skill.func_guardian_2>)
SHAMAN_1 = (0, '自己再生', [TriggerType.HEAL], <function Skill.func_shaman_1>)
SHAMAN_2 = (1, 'シャーマン２', [TriggerType.HEAL], <function Skill.func_shaman_2>)
